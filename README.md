## ArticleX
Convert any X (Twitter) long-form post to HTML, Markdown, or DOCX.

### Setup & Deploy
1. Fork this repo
2. Go to Settings -> Pages -> Source: GitHub Actions
3. Go to Settings -> Actions -> Read and write permissions
4. Push to main branch - auto-deploys via GitHub Actions
5. Change `base` in vite.config.ts to match your repo name

### Tech Stack
Vite · React · TypeScript · Tailwind CSS · Framer Motion · doc
